POLÍTICA DE PRIVACIDADE DE DADOS

Esta política descreve como a SPORTZ VIAGENS ESPORTIVAS LTDA ME., com sede na Av. Jose Cesar de Oliveira nº 181 – CJ 909, Vila Leopoldina, São Paulo / SP, CEP: 05317-000, inscrita no CNPJ/MF sob o nº. 22.981.030/0001-69 trata os dados pessoais.

Descreve também como esses dados são usados, compartilhados e como podem ser acessados, alterados ou excluídos. Este instrumento é classificado como uma medida administrativa voltada ao compromisso e a privacidade dos dados pessoais fornecidos por VOCÊ, para a execução de nossas atividades.

POR QUE A SPORTZ TRATA SEUS DADOS PESSOAIS (FINALIDADE)

A SPORTZ tem o compromisso com a transparência, a privacidade e a segurança dos dados de seus clientes, colaboradores e fornecedores durante todos os processos de tratamento de dados pessoais realizados.

Como parte das nossas atividades, necessitamos coletar dados pessoais de nossos clientes, colaboradores e fornecedores visando o fornecimento de nossos produtos e para melhor atendê los.

Com intuito de esclarecer o tratamento de dados pessoais pela SPORTZ, o entendimento das informações coletadas e como as utilizamos, tratamos, armazenamos ou excluímos, detalhamos a seguir nossa Política de Uso e Privacidade.

CONCEITOS BÁSICOS

Seguem abaixo algumas definições importantes dos principais conceitos envolvendo a Lei Geral de Proteção de Dados:

Termo Definição
Dado Pessoal Toda informação relacionada a uma pessoa natural que a identifique ou que, em conjunto com outros dados, permita identificá-la (identificável).
Tratamento Toda operação realizada com Dados Pessoais, como: coleta, produção, recepção, classificação, utilização, acesso, reprodução, transmissão, distribuição, processamento, arquivamento, armazenamento, eliminação, avaliação, modificação, transferência ou extração.
Titular Pessoa natural a quem se referem os Dados Pessoais que são objeto de tratamento.
Política Refere-se à presente Política de Privacidade e Proteção de Dados Pessoais, que estabelece as diretrizes para o uso de informações pela empresa.
Cliente Pessoa física ou jurídica que contrata produtos ou serviços
Fornecedor Pessoa física ou jurídica externa que provê insumos, produtos ou serviços necessários à operação.
Terceiro Pessoa física ou jurídica que presta serviços específicos para a SPORTZ, podendo ter acesso a dados para fins contratuais.
Controlador Pessoa jurídica ou física a quem competem as decisões fundamentais sobre o tratamento dos dados pessoais.
Operador Pessoa física ou jurídica que realiza o tratamento de dados pessoais seguindo estritamente as ordens e instruções do Controlador.

QUAIS DADOS PESSOAIS TRATAMOS

Ao se cadastrar ou demonstrar interesse pelos nossos serviços, serão fornecidos dados de identificação e dados cadastrais, tais como nome completo, número de um documento de identificação e/ou número do passaporte, telefone de contato, endereço de e-mail, data de nascimento, sexo, endereço, entre outros dados de identificação necessários para a contratação de nossos serviços.

Dados de crianças e adolescentes: Nós poderemos tratar os dados menores de idade para a prestação dos serviços. A SPORTZ se compromete a tratar tais dados pessoais para o melhor interesse da criança ou adolescente, e exclusivamente para a execução do contrato. Neste caso o responsável legal de cada um dos menores torna expresso o seu consentimento, no momento da inserção dos dados e assinatura da contratação, para o tratamento dos dados pessoais da criança e/ou adolescente, para a adequada prestação dos serviços contratados.

Documentação comprobatória em caso de rescisão/desistência: No caso da ocorrência de algum evento que impeça que o serviço seja usufruído, poderá ser necessário que haja o compartilhamento de documentos comprobatórios que poderão revelar dados sensíveis próprios ou de terceiros. Tais documentos podem incluir, mas não se limitar a, atestado de saúde, certidão de óbito, certidão de nascimento, dentre outros, a depender do caso concreto.

COMPARTILHAMENTO DE DADOS

A SPORTZ mantém cláusulas contratuais específicas para assegurar o correto tratamento dos dados pessoais, alinhadas com as leis e regulamentações brasileiras, previstas para o compartilhamento internacional de dados.

A SPORTZ não compartilhará dados pessoais de seus clientes, fornecedores e colaboradores, com uma terceira parte, exceto:

  1. Quando for necessário atender às obrigações legais ou regulatórias;
  2. Para cumprir contratos firmados
  3. Quando houver consentimento do titular de dados para tanto;
  4. Para pesquisa de crédito, quando necessário para processamento de pagamento, prevenção de fraudes, redução de risco de crédito ou outros serviços financeiros, na forma da lei;
  5. Quando houver necessidade de atuação de prestadores de serviços, haverá compartilhamento com terceiros para os fins descritos nesta política.

TRANSFERÊNCIA INTERNACIONAL DE DADOS

Em algumas situações, seus dados podem ser transferidos para fora do Brasil, sempre com garantias adequadas de segurança e conformidade com a LGPD.

QUAIS SÃO OS SEUS DIREITOS COMO TITULAR DE DADOS PESSOAIS?

O Titular de Dados poderá a qualquer momento, dentre outros:

  1. Solicitar informações, atualizar e retificar eventuais Dados Pessoais fornecidos;
  2. Solicitar a eliminação de quaisquer Dados Pessoais fornecidos, ressalvadas as hipóteses legais para a manutenção dos dados pessoais acima descritos;
  3. Revogar o consentimento para tratamento de Dados Pessoais, bem como para o compartilhamento de referidos dados pessoais a terceiros, nos termos desta Política de Privacidade; e
  4. Exercer quaisquer outros direitos previstos na legislação aplicável.

Em sendo o tratamento de Dados Pessoais essencial para a finalidade para as quais foram coletados, eventuais solicitações de eliminação de dados e/ou revogação de consentimento poderão resultar na exclusão de sua conta de acesso ou interrupção de quaisquer funcionalidades, sem aplicação de penalidade.

POR QUANTO TEMPO OS DADOS PESSOAIS SERÃO ARMAZENADOS?

Mantemos dados pessoais pelo tempo necessário para fornecer os produtos e serviços contratados e cumprir obrigações contratuais, legais e regulatórias, além de fiscais, de auditoria e contabilísticas, bem como para fins comerciais legítimos, obedecendo os prazos e critérios estabelecidos na legislação vigente.

COMO SEUS DADOS SÃO PROTEGIDOS?

Adotamos rigorosos padrões de segurança técnica e organizacional para salvaguardar os dados pessoais contra acessos não autorizados, perda, destruição ou alteração ilícita. Tais salvaguardas abrangem controles físicos, administrativos e digitais, dimensionados de forma proporcional à sensibilidade e ao volume das informações tratadas. O acesso aos dados é estritamente limitado a colaboradores autorizados, sob dever de confidencialidade, garantindo que o tratamento ocorra exclusivamente para as finalidades legítimas descritas nesta Política.
